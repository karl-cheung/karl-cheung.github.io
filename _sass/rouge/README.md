# rouge

## 生成

```shell
rougify style base16.monokai > _base16-monokai-syntax.scss
rougify style base16.solarized > _base16-solarized-syntax.scss
rougify style colorful > _colorful.syntax.scss
rougify style github > _github.syntax.scss
rougify style gruvbox > _gruvbox.syntax.scss
rougify style igorpro > _igorpro.syntax.scss
rougify style molokai > _molokai.syntax.scss
rougify style monokai > _monokai.syntax.scss
rougify style monokai.sublime > _monokai-sublime.syntax.scss
rougify style pastie > _pastie.syntax.scss
rougify style tulip > _tulip.syntax.scss
```

or

```shell
chmod 777 rouge.sh

./rouge.sh
```
